# Plant-Disease-Detection
<p>Plant Disease Detection is a digitalized method for the farmers in which he will upload the image of the plant leaf and detects the disease infection. Our system that is designed to be automated system is implemented using image processing technique using MATLAB. </p>
<p>This system will be beneficial for the farmers as well since it will facilitate for them the detection of the diseases because most of the farmers do not have a sufficient background and knowledge about monitoring the crops and dealing with the variety of diseases that could affect them.
</p>
<p>The system successfully extracted the infected area of the leaf and isolated the background and the green part of the leaf. Also, it shows whether the plant is healthy or unhealthy. </p>
